# 🖥️ Frontend Mini Projects

A collection of three responsive single-page websites built with **HTML**, **CSS**, and **JavaScript** as part of front-end development practice. Each project focuses on clean UI design, responsiveness, and real-world layout patterns.

---

## 📁 Projects

### 1. 🌐 [One Page Website](https://github.com/Bill-Earnest-Paul/one-page-website)
A versatile responsive landing page template built with HTML, CSS, and Bootstrap.

**Tech Stack:** HTML5 · CSS3 · Bootstrap  
**Highlights:**
- Fully responsive layout using Bootstrap grid
- Clean and modern UI suitable for portfolios or landing pages
- Cross-device compatibility (desktop, tablet, mobile)

---

### 2. 🧼 [Cleaning Service Website](https://github.com/Bill-Earnest-Paul/cleaning-server-website)
A business website for a cleaning service with an image slider and multi-section layout.

**Tech Stack:** HTML5 · CSS3 · jQuery · bxSlider  
**Highlights:**
- Smooth image slider using the bxSlider jQuery plugin
- Sections: Home, Services, About Us, Contact
- Responsive design for all screen sizes

---

### 3. 🛍️ [Online Shopping Website](https://github.com/Bill-Earnest-Paul/online-shopping-website)
A responsive e-commerce front-end with product listings and a contact footer.

**Tech Stack:** HTML5 · CSS3 · Media Queries  
**Highlights:**
- New Arrivals and Services sections
- Fully responsive using CSS media queries (no frameworks)
- Clean, modern shopping UI

---

## 🛠️ Tech Used Across Projects

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and content |
| CSS3 | Styling and layout |
| JavaScript / jQuery | Interactivity and DOM manipulation |
| Bootstrap | Responsive grid and components |
| CSS Media Queries | Custom responsiveness without frameworks |

---

## 🚀 How to Run Any Project

1. Click the project link above to open the repo
2. Clone it:
   ```bash
   git clone https://github.com/Bill-Earnest-Paul/<repo-name>.git
   ```
3. Open `index.html` (or `Home.html` for the shopping site) in your browser
